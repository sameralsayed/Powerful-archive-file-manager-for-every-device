# 📦 RARX – Smart Archive & File Manager Landing Page  
📄 A sleek, mobile‑first landing page for a compression and file manager–style app inspired by modern RAR tools.  

## ✨ Features  
⭐ Ultra‑clean dark UI that feels native on both mobile and desktop.  
📱 Fully responsive layout using Bootstrap 5, CSS3, and HTML5.  
🧩 Separate `index.html`, `style.css`, `script.js`, `README.md`, and `LICENSE` files (no folders).  
📂 Archive‑focused copy: create and extract RAR, ZIP, 7z, ISO, TAR, GZ, BZ2, XZ, ARJ, and more (text only, no engine).  
🛡️ Emphasis on encryption, repair tools, multi‑volume archives, and file management.  
🧪 Demo “Simulated archive” area wired with jQuery in `script.js`.  
🔍 SEO‑ready meta tags, Open Graph, and Twitter Card configuration.  
🎨 Custom gradients, glassmorphism cards, and iconography via Font Awesome.  
⚙️ Smooth scrolling and small UX touches implemented with jQuery.  
👨‍💻 Footer credit for SAMER SAEID with “Made by ♥️” and profile link.  

## 🗂 Project structure  
📄 `index.html` – Main landing page markup and SEO meta tags.  
🎨 `style.css` – Custom theme, layout, gradients, and responsive tweaks.  
🧠 `script.js` – jQuery interactions, smooth scroll, and simulated archive logic.  
📘 `README.md` – Overview, features, tags, and license note for GitHub.  
📜 `LICENSE` – MIT License text.  

## 🏗 Tech stack  
🌐 HTML5 for semantic, SEO‑friendly structure.  
🎨 CSS3 for modern styling, gradients, and responsive design.  
🧱 Bootstrap 5 for grid, layout, and components.  
💡 Vanilla JavaScript for core logic.  
🧩 jQuery for DOM helpers and smooth scrolling.  
🎭 Font Awesome for archive and file‑related icons.  

## 🔍 SEO & accessibility  
🧭 Meta description, keywords, and social preview tags configured in `<head>`.  
🧱 Semantic sections (`header`, `section`, `footer`) for better structure and crawling.  
♿ Focus outlines and contrast‑friendly dark palette for accessibility.  
⚡ CDN‑hosted assets to keep load times fast.

1️⃣ Download or clone the project files into a single directory (no subfolders).  
2️⃣ Open `index.html` in your browser to view the landing page.
